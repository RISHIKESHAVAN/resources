# Python for Data Science
The content of this module is based on the [Python for Data Science (PY0101EN)](https://courses.cognitiveclass.ai/courses/course-v1:Cognitiveclass+PY0101EN+v2/course/) course on Cognitiveclass by IBM.

## Topics Covered
* What is Python and why is it useful?
* The application of Python.
* How to define variables.
* Sets and conditional statements in Python.
* The purpose of having functions in Python.
* How to operate on files to read and write data in Python.
* How to use pandas, a must have package for anyone attempting data analysis in Python.
