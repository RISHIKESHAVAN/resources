# Data Visualization
The content of this module is based on the [Data Science: Visualization](https://www.edx.org/course/data-science-visualization) course on edX.
It consists of notes taken by me to my understanding of the topics taught in the course. This course covers the basics of data visualization and exploratory data analysis using the statistical programming language R.

## Topics Covered
* Data Types and Distributions
* Normal Distribution
* Smooth Density Plots
* Quantiles 
* QQ plots
* Percentiles and Box plots
* GG plot
