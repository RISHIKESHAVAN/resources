## Syllabus

- Understanding the  Dataset

- Data Formatting

- Binning

### Module 3 -  Exploratory Data Analysis
- Descriptive Statistics
- Basic of Grouping
- ANOVA

- Model Evaluation using Visualization

### Module 5 - Working with Data in Python
- Model  Evaluation

- Ridge Regression
- Grid Search 
- Model Refinement
