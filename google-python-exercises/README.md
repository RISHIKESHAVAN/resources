# Google Python Training

The contents of these folders are the (solved) Python exercises from [Google Python Class](https://developers.google.com/edu/python).

The exercises cover the following topics:

+ Strings
+ Lists
+ Tuples
+ Dictionaries
+ Regular Expressions
+ Working with files
+ Working with File paths and directories using `shutil` and `subprocess` modules.
+ Working with URL utilities
