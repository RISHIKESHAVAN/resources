# Section 2: Understanding Blockchain Technology

## Why to learn Blockchain?

The following companies primarily look for blockchain developers:

+ Startups (<https://angel.co/>)
+ Corporate Tech firms (<https://blocktribe.com/>)
+ Banks
+ Governmental Orgs
+ Professional Services Firms (KPMG, Price Waterhouse Coopers etc.)

Typically asked requirements for the jobs:

+ Code in various languages (Java and C++)
+ Comfortable with Large Data
+ Understand/Interest in Cryptography
+ Understand the Blockchain

## Web 3.0

Blockchain in general has been referred to being part of Web 3.0.

Web 3.0 refers to specifically the following key points:

+ Connective Intelligence
+ Connecting Data
+ Concepts Applications
+ Ultimately people
